Función principal para la carga de heightmaps Esta función intenta primero cargar el heightmap precompilado, si no lo encuentra lo compila y lo guarda.

Atención:
La ruta al archivo no debe contener extensión ya que se usa la función append para añadir la extensión que se requiera en cada caso.
Parámetros:
filename Ruta a la carpeta y nombre del archivo SIN EXTENSIÓN
h_buffer Puntero al heightmap a crear
texture Textura que se usará en el mapa
Devuelve:
0 -> Ok
1 -> Imagen de alturas (.pgm) no encontrado
2 -> Archivo de información (.txt) no encontrado
3 -> Fallo en el malloc
4 -> El archivo de alturas no es un PGM
